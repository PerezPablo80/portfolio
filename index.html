    Tendermeter

    The Tendermeter is an innovation development to meassure the tenderness of the meat, for the slaughterhouse industry.

    The development was done in Java with the SBC based on Raspberry Pi 4 and an LCD1602 connected using an API made in Python

    Initially we try using different hardwares, from Udoo v3, v8, Asus Thinkerboard and even a Rpi2, but because of a requirement and availiability, we ended up using an Raspberry Pi 4b and potentially an Pi 5.

    We show only the prototype because the final product is not finished yet, even though the development is.

    The project is due to be ready in early 2025

    The development, the design and architecture of both hardware and sotware was made by myself

    Ticket system

    During the last couple of years, I've been developing the ticket system used in Jade Informatica.

    The development was done using php (fat free framework or F3), mySql, html,CSS and js.

    The software is actually in use in Jade and has: Clients, devices associated to Clients, users and the service ticket

    The ticket can be validated with the client, using a cel phone that you can place your signature on it and delivers a signed PDF to both the client and the business

    If the ticket is not validated, you can still print a printable copy or send a PDF of it to the client.

    Whenever a service is finished, the system delivers an email to the business (the service is done by a technician)
