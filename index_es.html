    Tendermeter

    El Tendermeter es un desarrollo innovador de medición de terneza de la carne

    Dicho desarrollo fue realizado en Java, actualmente está hecho con un SBC Raspberry Pi 4b y una pantalla LCD1602 que se conecta con un desarrollo hecho en Python. Inicialmente se hicieron pruebas con Udoo v8, Udoo v3, Asus thinkerboard y hasta un Rpi2, pero por un requerimiento de velocidad y disponibilidad, se terminó optando por RPi4 y potencialmente 5.

    Dejamos un video del prototipo, dado que no se puede presentar el proyecto finalizado aún por un tema de que aún no está el dispositivo en el mercado.

    La proyección es que el mismo estará a principio de 2025

    Tanto el desarrollo como el diseño y arquitectura tanto de software como de hardware fue realizado por quien escribe

    App de Servicio técnico de Jade Informática

    Durante los ultimos años se trabajó en el desarrollo con php (framework fat free framework o F3), mysql, html, CSS, js el sistema que actualmente funciona en la empresa para tickets de servicios

    El mismo cuanta con un sistema de clientes, dispositivos asociados a los clientes, usuarios y los servicios propiamente dicho

    El servicio se puede validar con el usuario, haciendole firmar el mismo en cualquier celular con conexión a internet.

    Dicha validación habilita el envío de un mail al cliente, si el campo fue ingresado.

    Así mismo los servicios cargados nos llegan un mail con la indicación y posteriormente cuando se firma o finaliza, nos envía la información para proceder a facturar dicho servicio si corresponde. Los servicios son realizados por los técnicos.

    Sistema de órdenes para cliente privado

    Realizamos un sistema para un cliente privado de ordenes de compra

    El mismo guarda clientes, productos y pedidos previos, con el fin de permitirle a los vendedores tener los pedidos previos y los precios previamente ofrecidos

    El sistema fue almacenado en un servidor local, debido a limitaciones presupuestarias por parte del cliente.

    Para el desarrollo de este producto se utilizó: ReactJS, NodeJS con sequelize y express y de base de datos se usó Postgres

    No se pueden brindar fotos del producto debido a que es un proyecto privado. No obstante esto el desarrollo fue realizado 100% por quien escribe.
